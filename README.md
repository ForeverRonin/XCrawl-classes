# XCC Classes for Dungeon Crawl Classics in Foundry VTT

Expands the [Dungeon Crawl Classics system](https://github.com/cyface/foundryvtt-dcc/) for the [Foundry Virtual Tabletop](https://foundryvtt.com) to include X-Crawl Classics character sheets.

## Features

- **5 Core XCC Classes**: Hacker, Mystic, Scout, Techie, and Warrior
- **Mojo Tracking**: Full support for the Mojo resource system
- **Fame Tracking**: Integrated Fame progression and scoring
- **DCC Compatibility**: Built on the robust DCC system framework

## Installation

Manifest URL: https://raw.githubusercontent.com/ForeverRonin/XCrawl-classes/main/module.json

## Usage

1. Install the DCC system in Foundry VTT
2. Install this module using the manifest URL above
3. Create a new Actor and select one of the XCC class sheets
4. Track Mojo and Fame directly on the character sheet

## X-Crawl Classics Classes

- **Hacker**: Masters of technology and programming
- **Mystic**: Wielders of supernatural powers and mojo
- **Scout**: Specialists in stealth and reconnaissance  
- **Techie**: Engineers and technology experts
- **Warrior**: Combat specialists and tactical experts
