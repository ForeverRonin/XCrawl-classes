# Changelog

## [1.0.0] - 2024-12-19

### Added
- Complete conversion from MCC Classes to X-Crawl Classics Classes
- 5 core XCC classes: Hacker, Mystic, Scout, Techie, Warrior
- Mojo resource tracking system
- Fame progression/scoring system
- XCC-specific character sheet templates
- Compendium pack structure for XCC classes and equipment

### Changed
- Module ID changed from `mcc-classes` to `xcc-classes`
- All MCC references replaced with XCC
- Localization updated for X-Crawl Classics terminology
- CSS file renamed to xcc.css

### Removed
- All MCC class sheets (Healer, Manimal, Mutant, Plantient, Rover, Sentinel, Shaman)
- MCC-specific mutations and archaic alignment systems
- AI recognition mechanics
- Artifact check systems

### Technical
- Built on DCC Foundry system framework
- Compatible with Foundry VTT v13
- Maintains DCC system's stat framework integration
